# 도메인 그라운딩 — 한국 주식 범용 용어집 (시간 무관 · 누수 0)

> 라벨링 LLM 전용 보조자료. **딥러닝 모델 입력에는 들어가지 않음.**
> **종목 불문 범용** — 특정 종목 애칭·인물·테마·제품(예: 천슬라·젠슨황·머스크·FSD·HBM 등)은 일절 미포함.
> 그래야 TSLA·NVDA뿐 아니라 **새로운 종목에도 그대로 적용**된다(종목 특화 md는 주입 금지).
> 종목별 실적발표일 등 **날짜 의존 정보는 별도 '촉매 캘린더'로 분리**하고 이 파일엔 넣지 않는다.

[예측 vs 리액션 — 핵심 판정]

## 부록 (사람 참고용 — 주입 안 함)

### 방향 은어 상세
- **따상**: 신규상장주가 공모가 2배(따블)로 시초 형성 후 상한가 → 강한 상승 ▲. *(상한가 제도는 한국장 한정. 미국장 종목 댓글에서는 과장·비유.)*
- **양전/음전**: 전일 종가 대비 +로 전환=양전 ▲ / −로 전환=음전 ▼.
- **곱버스**: 곱하기+인버스. 지수 하락에 2배 베팅하는 ETF → 하락 기대 ▼.
- **숏스퀴즈**: 공매도 세력이 손실 회피로 강제 환매수 → 단기 급등 ▲.
- **설거지 / 폭탄돌리기**: 세력이 개미에게 고점 물량을 떠넘기고 빠지는 것 ▼.
- **데드캣(bounce)**: 급락 중 잠깐 반등. 추세는 여전히 하락 ▼.
- **개미털기**: 일부러 흔들어(급락 연출) 개미 손절 유도 후 다시 올림.
- **마진콜 / 반대매매 / 청산**: 신용·레버리지 거래에서 담보 부족 → 증권사 강제 매도. 패닉 국면 빈출, 극단적 하락 신호 ▼.
- **한강**: 극심한 손실·절망을 가리키는 밈. 라벨링상 **하락·절망 신호 ▼** 로만 분류(의미 상술 불필요).
- **셀온뉴스(sell on news) / 재료소멸**: 기대에 올랐다가 호재가 실제 발표되면 오히려 매도 → 하락 ▼. "소문에 사서 뉴스에 판다." 상승으로 단정하면 안 되는 대표 함정 ◆.

### 종목 종류 상세
| 종류 | 특징 |
|---|---|
| 대장주 | 해당 테마·섹터에서 시총·상징성 1등. 자금이 먼저 몰림 |
| 테마주 | 정치·신기술·이슈로 엮인 종목. 뉴스에 급등락, 변동성 큼 |
| 품절주 | 대주주 지분 높아 유통물량 적음 → 작은 수급에도 급변동 |
| 가치주/우량주 | 저PER·저PBR·안정 실적. 방어적, 변동 작음 |
| 성장주 | 매출·이익 고성장 기대. 밸류 높고 변동 큼 |
| 배당주 | 배당수익 목적. 배당락일 전후 수급 변화 |
| 보통주 vs 우선주 | 우선주는 의결권 없는 대신 배당 우대, 유통량 적어 변동 큼 |
| 자사주 | 회사가 자기 주식 매입(소각 시 주주가치↑) → 통상 호재 ▲ |

### 주의 (라벨링 시)
- **색을 거꾸로 읽지 말 것** — 빨강=상승, 파랑=하락. 영어권 학습 LLM의 최빈 실수.
- **종목 종류 용어 자체는 방향성이 아님** — 테마주라고 상승/하락이 아니다. 문맥의 동사·어조로 판단.
- **보유 행위 ≠ 방향 예측** — 물타기·불타기·존버·홀딩은 매매·보유 행위. 단독이면 예측_방향=없음 쪽.
- **완료형 동사·결과·정서는 리액션 우선** — "팔았다/물렸다/대박/멘붕"처럼 미래 단언이 없으면 시점관계=리액션.
- **조건부 예측은 방향은 주되 확신도를 낮게** — "뚫으면 간다"를 강한 예측으로 잡으면 Class 2·3이 부풀고, 무시하면 Class 0으로 샌다.
- **셀온뉴스 주의** — 호재가 곧 상승이 아니다(◆ 맥락의존).
